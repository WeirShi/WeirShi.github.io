github hexo 搭建个人技术博客

发表于 2017-10-02 | 分类于 前端 ， Blob

前言

前俩天刚搭建好这个博客，想着写的啥东西，今天就先把用Github+hexo搭建的过程记录一下吧。(虽然很多网上随便一搜就能出来很多😂)
我用的是Mac OSX系统搭建的，Windows的大同小异，小伙伴们对比着看吧。

开始

注册github账号

首先在Github上注册一个自己的账号吧，到GitHub官网，注册自己的账号，填好用户名，密码，邮箱，完成邮箱验证就开始耍吧！

创建github.io项目仓库

账号创建完，登录Github,创建一个项目仓库

仓库界面设置


基本设置(主题选择可以随意，因为我之后是用了hexo，会直接覆盖掉的)

选择好主题，提交保存好之后，就可以访问我们的github.io博客地址了。我们github的博客也就搭建好了，后面直接写文章就可以了。
github.io也可以指向我们自己购买的域名，我的域名还没有备案所以我就没有弄。。。。😂

使用hexo搭建

上面简单的用github搭建了简单的博客的地址，接下来就介绍一下使用hexo配合github搭建的博客

hexo环境搭建

  1. 安装node和git

安装好之后用node -v, npm -v, git version 检查一下吧！
完成之后就建立一个文件夹，名字自定义就好，然后cd进去，在Terminal里面输入命令

等待安装完成后，hexo v 检查是否安装成功，完成之后就可以使用了。

  1. 初始化
    输入 hexo init 命令开始初始化，会安装相关的依赖，等待完成后，输入 hexo s(hexo server的简写)，开始跑服务

接下来在浏览器里，输入http://localhost:4000 就可以看到我们初始化后的页面啦。。。。

到这里，hexo搭建的博客也就完成了，后面就是可以来写文章了。。。
搭建就写到这里吧，后面再写一点关于hexo的使用吧！
